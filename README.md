# Secure Cloud Infrastructure

## Objective
Secure a cloud infrastructure using Microsoft Azure by implementing identity management, backups, web application firewall (WAF), and public datasets.

## Services Used
- Microsoft Azure
- Azure Cloud Shell
- Application Gateway
- Web Application Firewall (WAF)

## Tasks Completed

### 1. IAM
Configured identity and access management.

### 2. Backup
Configured backup for infrastructure resources.

### 3. Web Application Firewall
Created and configured an Azure WAF Policy.
Enabled Prevention mode.
Applied managed security rules.

### 4. Public Dataset
Downloaded a public dataset and uploaded it to Azure Cloud Shell.
